Create navigational links within a website

There are several techniques to navigate within website. The semantic tag for navigation is the < tag. You can naviagate to your website by using relative addressing. For our examples, we will use the undeorder list tag (<ul>).

  • page to page within the same folder
  • page to page between folders
  • staying on the same current page
  • going to a specific place on the current page (on page navigation)

Navigational menus are placed in your <header> semantic area. They may also be place in your <footer> area in a sitemap. The new tag like other sematic tags should have a <hx> tag. HOWEVER, this header is optional. If you omit the header tag, you will receive a document outliner error: Untitled Selection. Until you are shown how to avoid this error message, it will be acceptable on your document outline.
